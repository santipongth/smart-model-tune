

# เปลี่ยนภาษาเริ่มต้นเป็นภาษาไทย

## สิ่งที่จะเปลี่ยน
เปลี่ยนค่า default language จาก English (en) เป็น Thai (th) เพื่อให้ผู้ใช้ที่เข้ามาครั้งแรกเห็นเว็บเป็นภาษาไทยทันที

## รายละเอียดทางเทคนิค

### ไฟล์ที่แก้ไข: `src/i18n/LanguageContext.tsx`
- แก้ฟังก์ชัน `getInitialLanguage()` ให้ return `"th"` แทน `"en"` เมื่อไม่มีค่าใน localStorage
- เปลี่ยนจาก 1 บรรทัด: `return "en";` เป็น `return "th";`

มีแค่นี้เลย -- แก้จุดเดียวแล้วทุกหน้าจะเริ่มต้นเป็นภาษาไทยโดยอัตโมมัติ ผู้ใช้ที่เคยเลือกภาษาไว้แล้วใน localStorage จะยังคงใช้ภาษาที่เลือกไว้เดิม
